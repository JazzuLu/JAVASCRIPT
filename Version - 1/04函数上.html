如何在使用函数时，向函数输入数据（参数）？

//num1和num2是函数的参数，定义参数时，不可以加var
function calculator(num1,num2){
     console.log(num1+num2);
}
//执行函数时，需要将数据传递给对应的参数
calculator(19,56);

注意：
1 在函数调用传入参数时，要确保数据的顺序正确
2 在函数调用传入参数时，要确保数据的类型正确
3 在函数调用传入参数时，要确保数据的内容正确

函数如何将处理结果返回？

return 关键字 ，用来让函数返回结果

如果函数没有return，那么执行完后，将返回什么？？    
undefined

return 要返回的内容;

举例：
function test(a, b, c){
     return a+b+c;
}
var s = test(5,6,7);
console.log(s);      //18
