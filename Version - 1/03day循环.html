循环的概念

循环就是重复做一件事 循环结构是程序中一种很重要的结构，其特点是给定条件成立时，反复执行某段程序，直到条件不成立为止。

while循环

语法：while（循环条件）{                
          语句；           
      }
注意： 1.在循环体内的代码被执行之前，就会对条件进行判断，只有条件成立，才会执行循环。因此，循环体内的代码有可能永远不会被执行。
      2.循环体中，应该有结束循环的条件，否则会造成死循环

while循环案例

例，用while循环求1到10的和
var i=1 ;
sum=0；
while(i<10){   
     sum=sum+i;   
     i+=1;
}

解析：在这个例子中，变量i开始时的值为1，每次循环都会递增1。而只要i的值小于10，循环就会继续下去

do-while

语法：          do{             
                     语句；         
               }while(判断);
do-while语句是一种后判断循环语句，即只有在循环体中的代码执行一次之后，才会判断是否满足循环的条件。换句话说，在执行判断之前，循环体内的代码至少会被执行一次。

continue
continue语句的功能： 只能在循环语句中使用，使本次循环结束，即跳过循环体中下面未执行的语句，接着进行下次是否执行循环的判断。
强调： 1.continue语句只能在循环体里
       2.对于while和do-while循环，

continue的作用

计算从1到10之间除了能被5整除的数的和？
var num=0;
var i=1;
while(i<=10){    
  if(i%5==0){        
       continue;      
      }     
  num+=i;
  i++;
}
alert(num);

for循环

用for循环求1~100的和
var  i,sum=0;
for(i=1;i<=100;i++){     
     sum=sum+i;
}
解析：for:循环初始条件
     i<=100:循环条件
     i++:循环变量增量

break

break语句的功能：        
在switch语句中使流程跳出switch结构        
在循环语句中使流程跳出当前循环 强调：       
1.如果已经执行break语句，就不会执行循环体中位于      break后的语句    
2.在多层循环中，一个break语句只能向外跳一层


案例：判断一个数是不是合数( 合数是指自然数中除了能被1和本身整除外，还能被其他数（0除外）整除的数。)


break和continue

和break一样，continue也是跳出循环。 区别在于continue是跳出当次循环，break是结束这个循环

死循环

概念：在编程中，一个无法靠自身的控制终止的循环称为“死循环” 死循环在系统的应用非常多，也非常重要，所有的应用系统都需要设置一个死循环来保证系统的正常运行，如果没有死循环，那么会一开机马上就关机，因为这个程序已经运行完毕，所以在系统开发中死循环有着极其重要的作用！（了解）

DEBUG工具

查找程序的错误过程，称为DEBUG 首先，给程序的某一行，设置断点 当程序运行到断点位置时，设置断点 当程序运行到断点位置时，暂停，等待指令 Step Over按钮，让程序可以一步一步的执行，从而观察整个程序的运行过程 这对于找出错误来说，是非常方便的 DEBUG是程序员的基本必备技能
